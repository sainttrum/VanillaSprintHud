# VanillaSprintHud

A simple mod that adds a HUD element showing whether sprint mode is currently toggled. It works best with toggle sprint, but also functions with hold sprint.

## Features
- Lightweight and simple
- Displays sprint status in the HUD
- Works with both **toggle sprint** and **hold sprint**
- Minimalist, vanilla-feeling design
- Customizable text via resource packs
